批处理基础

注释

  • REM 或 :: 用于添加注释。注释不会被执行。

显示文本

  • echo 显示文本。@echo off 可以关闭回显（即关闭命令本身的显示），但保留输出。

变量

  • set 用于定义变量，%variable% 用于访问变量值。

  • set /p 从用户输入中获取变量值。

条件语句

  • if 用于执行条件检查，可以用于字符串比较、数值比较和文件存在性检查。

    • /i：忽略大小写。

  • 比较操作符：

    • 字符串比较：==
    • 数值比较：equ（equal，等于）、neq（not equal，不等于）、lss（less than，小于）、leq（less than or equal to，小于等于）、gtr（greater than，大于）、geq（greater than or equal to)，大于等于

[!TIP]

当一个判断（循环或其他）语句之后的操作有多行时，可以用圆括号括起来表示一个作用域

循环

  • for 用于创建循环，用于遍历文件、字符串、命令输出等。%%表示的是脚本内部的变量

  • /L 参数用于创建计数循环。从start开始，一次增加step，到end结束

跳转与标签

  • goto 和 : 用于跳转到脚本中的指定位置。

调用其他脚本

  • call 用于从一个批处理文件中调用另一个批处理文件或子例程。

暂停执行

  • pause 用于暂停脚本执行，并提示用户按任意键继续。

退出脚本

  • exit 用于退出批处理文件，并返回到命令提示符。

常用命令

文件操作

  1. type：创建文件。

  2. copy: 复制文件。

    • /v：验证文件是否正确写入。

    • /y：覆盖现有目标文件时不提示。

    • /a：将文件标记为 ASCII 文本模式。

    • /b：将文件标记为二进制模式。

  3. del: 删除文件。

    • /f：强制删除只读文件。

    • /q：安静模式，删除时不提示确认。

    • /s：删除指定目录及所有子目录中的文件。

  4. move: 移动（重命）文件。

  5. xcopy: 复制文件和目录，包括子目录。

    • /i: 如果目标不存在，假设目标是一个目录。
    • /s：复制所有非空子目录。
    • /e：复制所有子目录，包括空的子目录。
    • /v：验证文件是否正确写入目标位置。
    • /y：覆盖现有目标文件时不提示。

目录操作

  1. dir:显示目录内容

    • /p：分页显示结果。
    • /w：以宽列表格式显示结果。
    • /s：递归显示目录及其所有子目录中的文件。
    • /b：使用简洁格式显示结果，仅显示文件和目录名。
    • /o：按指定顺序排序显示文件和目录。
      • N：按名称排序。
      • S：按大小排序。
      • E：按扩展名排序。
      • D：按日期/时间排序。
      • G：首先列出目录，然后是文件。
      • -：逆序排序。
    • /q：显示文件的所有者。
    • /a：显示具有特定属性的文件。
      • R：只读文件。
      • H：隐藏文件。
      • S：系统文件。
      • D：目录。
      • A：准备归档的文件。
      • L：符号链接。
    • /c：显示文件大小时包含千位分隔符（默认行为）。
    • /n：以默认格式显示文件名（与 /o 结合使用）。
    • /x：显示 8.3 短文件名。
    • /t：指定用于排序或显示的时间字段。
      • C：创建时间。
      • A：上次访问时间。
      • W：上次写入时间。

  2. mkdir: 创建目录。

  3. rmdir: 删除目录。

    • /S: 删除目录及其所有子目录和文件。
    • /Q: 安静模式，不提示确认。

  4. cd: 更改当前目录。

    • cd .. 返回上一级目录。

  5. 获取文件路径

    • %~f1：获取文件的完整路径。

    • %~d1：获取文件的驱动器号。

    • %~p1：获取文件的路径（不包括文件名）。

    • %~n1：获取文件名（不包括扩展名）。

    • %~x1：获取文件扩展名。

  1. if: 条件判断。

    • Example:

  2. for: 循环执行命令。

    • Example:

  3. goto: 跳转到指定标签。

    • Example:

  4. call: 调用另一个批处理文件或标签。

  5. exit: 退出批处理脚本。

    • exit /b: 退出子批处理文件并返回父脚本。

输入/输出

  1. echo: 输出文本。

    • echo. 输出空行。

  2. pause: 暂停执行并等待用户按键。

  3. set /p: 获取用户输入并赋值给变量。

字符串处理

  • 获取子字符串： :~ 用于从变量中提取子字符串。

  • 字符串替换： :<old>=<new> 用于替换字符串中的子串。

处理错误

  • 检查错误代码： 通过 if errorlevel 检查上一个命令的退出代码。

  • 设置退出代码： 使用 exit /b 设置批处理脚本的退出代码。

特殊符号

  • %：用于引用变量（如 %variable%）。

  • &：用于将多条命令放在同一行上执行。

  • |：管道符号，用于将前一命令的输出作为下一命令的输入。

  • > 和 >>：重定向符号，用于将输出写入文件（覆盖或追加）。

  • <：输入重定向，用于将文件内容作为命令的输入。

  • ^：转义符号，用于转义特殊字符。

  • && 和 ||：条件执行符号，分别表示“如果成功则执行”和“如果失败则执行”。

# 快捷键

## 常用快捷键

| 快捷键                                                     | 功能                     | 备注 |
| ---------------------------------------------------------- | ------------------------ | ---- |
| <kbd>Ctrl</kbd><kbd>p</kbd>                                | 搜索文件                 |      |
| <kbd>Ctrl</kbd><kbd>，</kbd>                               | 打开设置                 |      |
|                                                            |                          |      |
| <kbd>Ctrl</kbd><kbd>Shift</kbd><kbd>p</kbd>                | 打开命令面板             |      |

# 使用技巧

## 忽略搜索文件

在`settings.json`文件中增加以下代码：

```json
{
    "search.exclude": {
        "**/Output": true,
        "**/.git": true,
        "**/.vscode": true,
        "**/Project": true
    }
}
```

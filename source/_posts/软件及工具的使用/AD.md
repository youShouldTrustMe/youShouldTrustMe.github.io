## PCB

|                快捷键                 |   功能   |
| :--------------------------------: | :----: |
|            <kbd>2</kbd>            | 转换为2D图 |
|            <kbd>3</kbd>            | 转换为3D图 |
|    <kbd>v</kbd> + <kbd>b</kbd>     | 翻转PCB图 |
| <kbd> CTRL</kbd> + <kbd>鼠标左键</kbd> | 跟踪线路走向 |
|                                    |        |



> [!TIP]
>
> 注意，所有的快捷键都在英文状态下执行



# 查找

在原理图中：

1. 知道其中一个信号在哪
   1. 在一个文件中查找：按住<kbd>ALT</kbd>，鼠标左键点击信号
   2. 鼠标右键网络标号，点击Find Similar Objects，将text改为same即可。

2. 不知道网络标号在哪
   1. <kbd>CTRL</kbd> + <kbd> F</kbd> 输入网络标号，点击搜索
